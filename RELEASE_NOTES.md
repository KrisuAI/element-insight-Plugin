## üöÄ **Getting Started**

### **Quick Installation**
1. **Install Plugin**: From IntelliJ Plugin Manager or download JAR file
2. **Install Playwright**: Use `Tools ‚Üí Element Insight: Install Playwright`
3. **Install Browsers**: Use `Tools ‚Üí Element Insight: Install Browsers`
4. **Open Tool Window**: Use `Ctrl+Alt+E` or `Tools ‚Üí Element Insight: Open Element Insight`
5. **Start Scanning**: Enter URL and begin element analysis

### **First Steps**
- **Scan a Website**: Test with a simple website to understand the workflow
- **Export Results**: Try different export formats (JSON, CSV, code)
- **Explore Features**: Use keyboard shortcuts and explore the UI
- **Read Documentation**: Review help system and documentation
